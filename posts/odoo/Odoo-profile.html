# Odoo @profile 性能测试

# odoo rst 文档

Another profiler can be used to log statistics on a method:

    from odoo.tools.profiler import profile
    [...]
    @profile
    @api.model
    def mymethod(...):

The statistics will be displayed into the logs once the method to be analysed is completely reviewed.
