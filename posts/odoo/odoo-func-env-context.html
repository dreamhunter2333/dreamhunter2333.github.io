context 中内容会在下次前端触发后重置

获取 context 内容 ​

python
self.env.context.get('demo'):

增加 context 内容 ​

python
context = dict(self.env.context, demo=True)
self.env = self.env(context=context)

python
self.with_context(demo=True).demo_func()
